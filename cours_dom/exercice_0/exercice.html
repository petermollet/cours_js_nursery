Exercice 0

Le but de cet exercice est de simuler une connection pour afficher un message à l'utilisateur selon s'il est connecté ou non. Grâce à un boolean 'estConnecte' (dans le fichier exercice.js) on voudra modifier le message dans la balise h2 ci-dessous. - Si connecter, afficher en bleu: 'Bonjour et bienvenue sur votre compte !' - Si pas connecter, afficher en rouge: 'Veuillez vous connecter à l'application avant de continuer
